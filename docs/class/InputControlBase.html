Class: InputControlBase

Defined in: app/tamarind/controls/InputControlBase.coffee
Inherits: UIComponent

Overview

Base class for input editors. Instances of these classes encapsulate the controls used to edit the values of inputs, and the classes themselves contain metadata e.g. default values
